## 1. Philosophy: Descriptive + Prescriptive

### The Unified Approach

| Approach | What It Does | When to Use |
|----------|-------------|-------------|
| **Descriptive** | Tells the AI *what to feel like* — style, mood, atmosphere | Initial concepts, creative exploration |
| **Prescriptive** | Tells the AI *what to do physically* — lighting ratios, focal lengths, material properties | Production output, consistent results |

**The Framework:**

```
Descriptive (WHO/WHAT)          Prescriptive (HOW EXACTLY)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Subject + Style + Mood      →   Physics Parameters + Technical Specs
┌─────────────────────┐        ┌─────────────────────────────┐
│  8-Layer Template   │   +    │  Physics-First Specs        │
│  Design Vocabulary  │        │  Lighting Ratios           │
│  Animal Thinking    │        │  Optical Parameters        │
│  Mode Chains        │        │  Material Properties       │
└─────────────────────┘        └─────────────────────────────┘
                    ↓
            Professional Output
```

### When to Use Each

| Task | Approach | Example |
|------|----------|---------|
| Concept exploration | Descriptive first | "Dreamy, ethereal portrait" |
| Production output | Add prescriptive | + "Rembrandt 4:1, 85mm f/1.4, SSS 1.2mm" |
| Brand consistency | Prescriptive foundation | Start with physics, add descriptive |
| Creative breakthrough | Descriptive + Dolphin mode | "What if gravity worked sideways?" |

| Ratio | Stops | Effect | Best For |

| 2:1 | 1 | Flat, even | Beauty, corporate |
| 3:1 | 1.5 | Natural modeling | Standard portrait |
| 4:1 | 2 | Dramatic | Editorial, leadership |

| Temp | Light Source | Mood |
|------|--------------|------|
| 2700K | Candlelight | Intimate, warm |
| 3200K | Tungsten | Cozy, warm |
| 4500K | Golden hour | Flattering |
| 5000K | Noon daylight | Balanced, natural |
| 5600K | Overcast | Cool, clinical |
| 6500K | Blue sky | Cold, dramatic |

### 3.3 Material Properties

#### Skin Rendering (Subsurface Scattering)

| Skin Type | SSS Depth | Prompt Token |
|-----------|-----------|-------------|
| Caucasian | 1.0–2.5mm | `SSS 1.2mm, pink/orange scatter` |
| Asian | 1.0–2.0mm | `SSS 1.0mm, yellow/orange` |
| African | 1.5–3.0mm | `SSS 2.0mm, red/orange` |

**Anti-Waxy Checklist:**
```
✓ visible pores
✓ fine vellus hair
✓ subsurface scattering
✗ plastic skin
✗ airbrushed
✗ poreless
```

#### Material IOR (Fresnel Reflectance)

| Material | IOR | Prompt Token |
|----------|-----|-------------|
| Skin (dry) | 1.38 | `Fresnel 0.028` |
| Skin (wet) | 1.40 | `Fresnel 0.03` |
| Glass | 1.52 | `Fresnel 0.04` |
| Water | 1.33 | `Fresnel 0.02` |

#### Roughness Values

| Value | Effect | Materials |
|-------|--------|-----------|
| 0.0–0.1 | Mirror-like | Polished metal, wet surfaces |
| 0.1–0.3 | Glossy | Satin, polished wood |
| 0.3–0.5 | Semi-gloss | Leather |
| 0.5–0.7 | Matte | Paper, wool |
| 0.7–1.0 | Diffuse | Felt, rough concrete |

## 6. Complete Workflow: Pre-Production to Refinement

### Phase 1: Pre-Production

```
□ Define objective (one sentence)
□ Identify audience and platform
□ Select aesthetic keywords
□ Determine constraints
```

### Phase 2: Physics Specification

```
□ Choose lighting pattern (Rembrandt, Butterfly, etc.)
□ Set lighting ratio (2:1 to 8:1)
□ Specify Kelvin temperature
□ Select focal length
□ Set aperture
□ Define material properties (SSS, Fresnel, roughness)
□ Plan advanced phenomena (1-2 max)
```

### Phase 3: Platform Selection

```
□ Match tool to use case:
  - Midjourney: Artistic, cinematic
  - DALL-E 3: Quick prototyping
  - FLUX: Multi-image, speed
  - Ideogram: Typography
  - Stable Diffusion: Technical control
```

### Phase 4: Prompt Construction

```
# Combine Descriptive + Prescriptive

DESCRIPTIVE:
[AESTHETIC] + [SUBJECT] + [MOOD]

PRESCRIPTIVE:
[Lighting ratio] + [Kelvin] + [Focal length]
[Material properties] + [Camera specs]

EXAMPLE:
"Editorial portrait" (descriptive) +
"85mm f/1.4, Rembrandt 4:1, SSS 1.2mm" (prescriptive)
```

### Phase 5: Generation + Iteration

```
□ Generate with seed locked
□ Analyze against physics spec
□ Adjust ONE parameter at a time
□ Iterate until acceptable
□ Document successful parameters
```

### Phase 6: Post-Processing

```
□ Check resolution (4K minimum for print)
□ Apply color grading
□ Inpaint any artifacts
□ Verify against quality checklist
□ Export for final use
```

### Template 1: Executive Portrait

```
[SUBJECT]: Executive portrait, male/female, 40s-50s
[DESCRIPTIVE]: Sharp jawline, confident gaze, authoritative presence
[PRESCRIPTIVE]:
- Rembrandt lighting, 4:1 ratio, 5600K key
- 85mm f/1.4, full-frame sensor
- SSS 1.2mm, visible pores, vellus hair
- Fresnel 0.028, matte wool suit
- ARRI ALEXA 65 color science
[NEGATIVE]: plastic skin, waxy, poreless, cartoon
[OUTPUT]: --ar 4:5 --style raw --seed [固定]
```

```
[SUBJECT]: [Product name] on [surface]
[DESCRIPTIVE]: Luxury, clean, commercial grade
[PRESCRIPTIVE]:
- Ring light 5600K, 2:1 ratio
- 90mm tilt-shift, f/11
- Anisotropic reflections
- Fresnel 0.04 (glass/metal)
- 4K minimum resolution
[NEGATIVE]: Shadows, reflections, artifacts
[OUTPUT]: --ar 4:3 --quality 2 --repeat 3
```

## 8. Quality Checklists

### Pre-Generation Checklist

| Category | Item | Priority |
|----------|------|----------|
| **Objective** | Goal clear in one sentence | Critical |
| **Subject** | Primary subject specified | Critical |
| **Lighting** | Pattern + ratio + Kelvin defined | High |
| **Optical** | Focal length + aperture set | High |
| **Material** | SSS + Fresnel + roughness specified | Medium |
| **Platform** | Correct syntax for tool | High |
| **Negative** | Artifacts excluded | High |

### Post-Generation Checklist

| Check | Validation |
|-------|------------|
| **Subject Clarity** | Main subject identifiable immediately? |
| **Anatomy** | Hands, eyes, proportions correct? |
| **Skin** | Pores visible? No plastic/waxy look? |
| **Lighting** | Shadows match specified ratio? |
| **Bokeh** | Shape matches lens/focal length? |
| **Resolution** | Sufficient for final use? |
| **Artifacts** | Any unwanted elements to inpaint? |

### The "Zero-Waxy" Audit

```
□ Skin shows visible pores at 100% zoom
□ Vellus hair visible on skin
□ Subsurface scattering evident (warm translucency)
□ Lighting ratio matches specification
□ Bokeh shape matches lens type
□ No plastic or airbrushed appearance
□ Eyes have catchlights
□ Hands have correct number of fingers
```
